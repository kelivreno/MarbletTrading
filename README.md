# Marble Trading

A simple betting game where the player starts with $1000 and wagers money on randomly drawn marbles.

Each round, a marble is drawn from a bag containing different colors, and the outcome determines how much the player wins or loses.

## Game Rules

The bag contains:
- 4 Green marbles
- 3 Red marbles
- 1 Black marble
- 1 White marble

Marbles are replaced after each round.

### Marble Outcomes
- Green → Win your bet amount
- Red → Lose your bet amount
- Black → Win 10× your bet
- White → Lose 5× your bet

## Concepts Used
- `if` statements
- `for` loops
- `while` loops
- `random` module

## How to Run
1. Run the Python file
2. Enter number of rounds
3. Enter your bet each round
4. Try not to go broke 😅
